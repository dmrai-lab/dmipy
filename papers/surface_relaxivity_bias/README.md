# Surface relaxivity biases diffusion and relaxometry microstructure estimates

Reproducibility package for the manuscript

> **Surface relaxivity biases the intra-axonal signal fraction and the myelin water fraction**

Surface relaxivity and time-dependent diffusion are two readouts of the same wall
collisions on one substrate. The transverse rate that microstructure imaging fits is a
bulk rate plus a surface rate `ρ·(S/V)`; intra- and extra-axonal water carry different
`S/V`, so their `T₂` differ and any compartment estimate that normalises by a
TE-weighted `b=0` is biased. This package contains the manuscript source and every script
needed to regenerate the figures.

- **Preprint:** arXiv (ID to be added on acceptance).
- **Engines:** [`dmipy-fit`](https://github.com/dmrai-lab/dmipy-fit) (analytical inverse) and
  [`dmipy-sim`](https://github.com/dmrai-lab/dmipy-sim) (forward Monte Carlo).

## Layout

```
main.tex, sections/*.tex, refs.bib, main.bbl   LaTeX source
main.pdf                                        compiled manuscript
appendix_estimator_comparison.py                appendix estimator benchmark (root: writes into figures/)
figures/fig_*.py                                one script per figure
figures/real_substrate.py, gather_real_substrates.py   real-SEM substrate helpers
figures/*_data.npz                              cached results so figures replot without a GPU
figures/rat_sem_crop*, figures/_real_xsec/      segmented rat-SEM cross-sections (see Data)
figures/*.pdf                                   the figures as they appear in the manuscript
```

## Build the PDF

```bash
pdflatex main && bibtex main && pdflatex main && pdflatex main
```

The shipped `main.bbl` also lets `pdflatex main` (twice) build it without running bibtex.

## Reproduce the figures

Install the two engines (both public):

```bash
pip install dmipy-fit dmipy-sim
```

Each figure is a standalone script. Those with a bundled `*_data.npz` replot instantly with
`--plot-only`; without that flag they recompute (some Monte Carlo runs prefer a GPU).

```bash
# analytical / instant (no engine needed):
python figures/fig_fintra_bias.py                     # f_intra bias vs packing, TE, calibre
python appendix_estimator_comparison.py               # per-collision estimator benchmark (~1 min CPU)

# replot from cached results (instant, no engine):
python figures/fig_mwf_sv_bias.py       --plot-only   # MWF vs S/V
python figures/fig_cpmg_mwf_engine.py   --plot-only   # signal / CPMG decay
python figures/fig_demyelination_bias.py --plot-only  # demyelination trajectory

# recompute with the engines (drop --plot-only; GPU recommended for the MC panels):
python figures/fig_crossing_mc.py        # isolated-axon wall-counting MC (use --quick on CPU)
python figures/fig_smt_fintra.py         # runs the SMT fits (dmipy-fit, ~1 min CPU)
python figures/fig_substrate_cross_section.py   # real + synthetic substrate cross-sections
```

Point `DMIPY_SIM_PUBLIC` / `DMIPY_FIT_PUBLIC` at a source checkout to override the installed
packages; otherwise the scripts import whatever is on the path.

## Data

`figures/_real_xsec/` and `figures/rat_sem_crop.png` are manual axon/myelin segmentations of
rat spinal-cord scanning electron microscopy from the **AxonDeepSeg** dataset
(Zaimi et al., 2018), redistributed here under its original license for reproducibility.
All other inputs (Gamma-distributed synthetic packs, biophysical constants) are generated by
the scripts and the engines.
